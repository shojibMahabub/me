The first VAPT report we saw had a lot of red. Weak encryption in a handful of places, access-control gaps in admin endpoints, a login flow that revealed more than it should.

We triaged by risk, fixed the class instead of the case, collapsed role-spaghetti into a single policy matrix, and logged the boring stuff so the pentester could see that things were actually happening.

The playbook is written up here:

- [Zero high-sev VAPTs. A playbook.](/blog/zero-high-sev-vapts/)

- Zero high-severity findings in the follow-up report.
- Security stopped being a fire drill and started being table stakes.
