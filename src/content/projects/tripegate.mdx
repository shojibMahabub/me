Tripegate was my first taste of real cloud infrastructure: multi-tenant RBAC, S3-backed document storage, IAM policies with least privilege, and a pile of third-party travel APIs that disagreed on everything.

Highlights from that run:

- Integrated 50+ travel and expense APIs; built reusable libraries to keep the blast radius small when any one of them shifted.
- Built secure multi-tenant SaaS modules using RBAC and policy-driven access controls.
- Deployed cloud infrastructure with AWS security best practices (IAM, SGs, NACLs).
- Configured S3-based storage workflows improving durability and data management.

It's the project where I stopped being afraid of infrastructure.
